# Mobile Sales Dashboard - Power BI 

Project Overview  

This project presents an interactive Mobile Sales Dashboard built using Power BI. The dashboard provides insights into sales performance, customer behavior, transaction trends, and brand-wise performance across different cities and time periods. 

Objectives
- Analyze mobile sales performance. 
- Track total revenue, quantity sold, and transactions.
- Compare sales across brands and mobile models.
- Understand customer ratings and payment preferences.
- Identify monthly and daily sales trends.

Tools Used
- Power BI
- Power Query
- DAX
- Microsoft Excel

Key Metrics  
- Total Sales: 769M
- Total Quantity Sold: 19K
- Total Transactions: 4K
- Average Sales Value: 40K
-
Dashboard Features  - Interactive slicers for Brand, Mobile Model, Payment Method, and Day Name. - Monthly sales trend analysis. - Brand-wise sales comparison. - Customer rating distribution. - Payment method analysis. - Day-wise sales performance. - City-wise sales analysis. 

Key Insights  - Apple and Samsung contributed significantly to overall sales. - UPI, Debit Card, Credit Card, and Cash payments were widely used by customers. - Sales showed fluctuations across different months, highlighting seasonal trends. - Customer ratings indicate generally positive satisfaction levels.
